## Замечания по переводу

Несколько тем имеют конкурирующие переводы в рунете — официальный маппинг выше:
- **Command** = Распорядитель (не «Командование»)
- **Belief** = Убеждение (не «Вера»)
- **Input** = Вклад (не «Накопление»)
- **Intellection** = Мышление (не «Мыслитель»)
- **Learner** = Ученик (не «Обучаемость»)
- **Relator** = Отношения (не «Общение»)
- **Connectedness** = Взаимосвязанность (не «Взаимосвязь»)
- **Adaptability** = Приспособляемость (не «Адаптивность»)
- **Includer** = Включенность (не «Включение»)
- **Futuristic** = Будущее (не «Футурист»)
- **Ideation** = Генератор идей (не «Генерация идей»)
- **Arranger** = Организатор (не «Распорядитель» — это Command)
