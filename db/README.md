# CliftonStrengths Theme Database

Источник: gallup.com/cliftonstrengths/en/themes (официальные страницы, апрель 2026)

34 файла в `themes/` — по одному на каждую тему. Структура каждого файла:
- **Definition** — официальное определение Gallup
- **Description** — развёрнутое описание + самоописания носителей темы
- **Working With** — как работать/управлять человеком с этой темой
- **At Their Best** — в каких условиях тема раскрывается максимально
